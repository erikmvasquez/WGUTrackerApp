WGU Tracker App

The WGU Tracker App is a mobile application built with .NET MAUI and SQLite to help WGU students organize their academic progress. It allows users to create and manage terms, courses, and assessments, with detailed views for instructors, notes, and deadlines.

Option 1: Install APK on an Android Device

  1. Download the APK file from the button above.
  2. On your Android device, go to Settings and click on Security (or Privacy) and enable “Install unknown apps” for the browser or Files app you will use.
  3. Open the downloaded WGUTrackerApp-v0.1.0.apk file from your Downloads folder.
  4. Tap Install.
  5. After installation, open the WGU Tracker App from your app drawer.
  6. Create an account on the login screen, then begin adding terms, courses, and assessments.

Option 2: Run the App from Source in Visual Studio

  1. Clone the GitLab repository:
    git clone https://gitlab.com/wgu-gitlab-environment/student-repos/evasq56/d424-software-engineering-capstone.git
  2. Open WGUTrackerApp.sln in Visual Studio 2022 (or newer).
  3. Ensure you have the following installed:
    • .NET 8 SDK
    • Visual Studio workload: .NET Multi-platform App UI development (.NET MAUI)
    • Android Emulator
  4. Set the startup project to WGUTrackerApp.
  5. Select an Android emulator (e.g., Pixel 7 Pro - API 35) from the toolbar.
  6. Click Run to build and deploy the app.

Troubleshooting

  • App not installed: Uninstall any previous version of the app, then try again.
  • Emulator install fails (ABI mismatch): Your Visual Studio emulator is usually x86_64. If the APK is arm64-only, it won’t install. Either create an ARM emulator or rebuild an x86_64 APK in Visual Studio.
  • Device not detected in Visual Studio: On your phone, enable Developer Options: USB Debugging, then reconnect and accept the USB debug prompt.
  • Still having issues? You can always run the app directly from source in Visual Studio using the steps above, which automatically builds and installs the correct version for your emulator/device.
