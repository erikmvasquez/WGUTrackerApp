WGU Tracker App

The WGU Tracker App is a mobile application built with .NET MAUI and SQLite to help WGU students organize their academic progress. It allows users to create and manage terms, courses, and assessments, with detailed views for instructors, notes, and deadlines.

Run Locally in Visual Studio

  1. Open WGUTrackerApp.sln in Visual Studio 2022 (or newer).
  2. Ensure you have the following installed:
    • .NET 8 SDK
    • Visual Studio workload: .NET Multi-platform App UI development (.NET MAUI)
    • Android Emulator
  3. Set the startup project to WGUTrackerApp.
  4. Select an Android emulator from the toolbar.
  5. Click Run to build and deploy the app.
